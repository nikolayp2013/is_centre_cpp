Окно начальных установок

Для входа в окно начальных установок Вам необходимо запустить программу до нашего логотипа. В правом верхнем углу есть кнопка "Установки". Щелкните на нем мышкой. Откроется окно начальных установок программы. В нем Вы увидите следующие закладки:

Система БД:
Настройка системы управления базой данных. Ее перенастройка должна согласовываться с администратором сети.

Резервное копирование:
Устанавливает число дней после последнего сеанса связи после которого система автоматически сделает резервное копирование ваших данных на сервер. Необходимо в случае потери вашей базы.

Настройка удаленного доступа:
Тип связи:
- Использовать параметры удаленного соединения: При установке соединения, все параметры (телефон, логин, пароль) будут использоваться те, что установлены в удаленном соединении.
- Использовать собственные настройки: При сеансе связи будут использоваться те настройки, которые указаны в данном окне.
- Не использовать удаленное соединение: Применяется в случае, если Ваш компьютер через локальную сеть непосредственно подключен к Internet.

Параметры по умолчанию:
- Использовать динамический фильтр. Включить если ваш компьютер имеет память не менее 32 Мегабайт. Тогда каждая выбранная позиция в фильтре сразу будет отрабатываться. Если "галочка" не установлена, то фильтр будет включать только по кнопке "Ок" в окне фильтра.
- Проверка адреса: - установите, если необходима проверка адреса вновь введенного объекта, на предмет дублирования.
- Работа с филиалами: - установите, если желаете в "своем" прайс-листе видеть объекты своих филиалов или группы агентств.
- Использовать общие кадры в филиалах: - Позволяет вести единый кадровый список только из головного филиала, и рассылка его в филиалы. При этом ввод новых агентов в филиалах недопустим. Возможна только корректировка. Корректируемая запись в филиале после сеанса связи попадает в главный офис, где так-же может корректироваться и направляться назад, в филиал.
- Использовать подачу рекламы для всех филиалов: - позволяет отмечать и генерировать рекламу с главного офиса для своих филиалов. Применяется в случае отсутствия ответственного лица за подачу рекламы в филиале.
