Для поиска агентов Вы можете воспользоваться кнопкой «Запрос». После ее нажатия перед Вами откроется окно запроса по кадрам.

В этом окне Вы можете составить достаточно сложный запрос по различным параметрам. Каждый запрос состоит из набора условий, объединенных по «и». Условие, в свою очередь, состоит из параметра, операции и значения. Допустим, Вам надо найти всех Ивановых старше 30 лет. Тогда Вы можете сформировать следующий запрос: «ФИО равна Иванов И Дата рождения меньше 01/01/1973». Нажав на кнопку «ОК» Вы можете выполнить введенный Вами запрос.

Введенный запрос Вы можете сохранить в информационной базе для дальнейшего использования. Для этого, введя запрос, нажмите кнопку «Записать». Программа спросит Вас об наименовании записываемого запроса и потом занесет его в информационную базу. Для использования уже записанного запроса выберите его по имени из выпадающего списка и нажмите кнопку «Загрузить». Для удаления сохраненного запроса выберите его и нажмите кнопку «Удалить».

Снятие флажка «Использовать запрос» выключает использование созданного Вами запроса.
