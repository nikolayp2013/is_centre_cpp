Панель инструментов состоит из основной и дополнительной.

Основная панель.

Основная панель состоит из следующих блоков:

  • выбор раздела недвижимости;
  • выбор прайс-листа;
  • работа с карточкой (добавление, просмотр, удаление, обновление);
  • выбор формы отображения прайс-листа;
  • кнопка обмена прайс-листами (отправка своих и получение чужих объектов, отправка СМИ);
  • информационные окна;
  • список клиентов;
  • смена пользователя;
  • помощь.

Дополнительная панель.

Дополнительная панель состоит из следующих блоков:

  • выбор предложения или спроса;
  • печать прайс-листов;
  • фильтр по типу договора (эксклюзивный, открытый, нет договорных отношений);
  • рекламация на объект;
  • экспорт в Excel;
  • ссылка на сайт нашей компании.

В программе существуют 13 разделов по предложению и 6 по спросу.
