Для удаления агента из системы Вы должны найти его в таблице персонала, установить курсор на его запись и нажать на кнопку «Удалить». Перед Вами откроется окно удаления агента.

В данном окне Вы можете произвести два действия:

  1. Удалить агента и все его объекты из информационной базы. Для этого нажмите на кнопку «Удалить».
  2. Удалить агента и передать его объекты другому агенту. Для этого выберите необходимого агента, которому надо передать объекты, из списка. Для упрощения поиска агентов необходимого агента можно найти по его рекламному телефону, для чего его телефон надо ввести в соответствующее поле и нажать клавишу «Enter». После выбора нового агента нажмите кнопку «Передать». После нажатия все объекты удаляемого агента будут переданы выбранному агенту.
