Верхнее контекстное меню программы состоит из следующих пунктов:

База:

  • Города - выбор города
  • Печать- печать прайс-листов
  • Выход

Карточка:

  • Добавить - добавление карточки объекта в прайс-лист
  • Просмотр - просмотр карточки объекта
  • Удалить - удаление карточки объекта
  • Обновить - обновление даты корректировки объекта
  • Сменить статус - перемещение объекта между внутренним, внешним и архивным прайс-листами

Прайс-лист:

  • Свой внешний - объекты, которые будут доступны другим абонентам сети
  • Свой внутренний - объекты, которые Вы хотите скрыть от остальных членов сети
  • Объединенный - объекты других абонентов сети и Вашего внешнего прайс-листа
  • Архивный - Ваши удаленные объекты

Вид:

  • Агентская форма - основная форма для работы агента
  • Клиентская форма - форма в которой отсутствует служебная информация (фирма, № карточки, дата, и.т.д.)
  • Для служебного пользования(ДСП) - в основном служит для подачи рекламы в СМИ

Метки:

  • для СМИ - метки для подачи объявления в печатные издания
  • для печати (обновления, удаления)

Статистика:

  • По кол-ву объектов у агентов
  • По кол-ву объектов у фирм
  • Список агентов в других фирмах
  • Реестр абонентов

Сервис:

  • Подача объявлений в СМИ
  • Кадровый модуль - список агентов (сотрудников) в программе
  • Информационные окна - полезная информация
  • Клиенты - список клиентов
  • Искать карточку по №
  • Редактирование групп - создание групп компаний (абонетов системы) для фильтрации объектов
  • Экспорт в Excel
  • Оформление
  • Установки
